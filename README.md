# Bouncemasters

A fun bouncing game built with PixiJS.

## How to use

Run a local web server:

```bash
python -m http.server 8000
```

Or use any other web server of your choice.

Then open your browser to `http://localhost:8000`

## Features

- Engaging bouncing gameplay
- Local progress persistence using browser localStorage
- Audio support with user gesture unlock
- Mobile-friendly controls
